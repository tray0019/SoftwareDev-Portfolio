Poll-Vote-App

Live Demo Source Code

Overview

A lightweight poll/voting app built with Spring Boot (REST), Spring Data + MySQL, and an Angular frontend. Users can create polls with multiple options, vote, and see totals update immediately.

Architecture

Backend (Spring Boot)
  • Controller exposes /api/polls (create, list, get by id) and /api/polls/vote (cast vote).
  • Service validates vote index, increments the selected option's voteCount, and persists.
  • Repository uses Spring Data JPA for persistence.
  • Model
    • Poll (id, question, options)
    • OptionVote (optionText, voteCount) as @Embeddable via @ElementCollection
  • CORS
    • Global CorsFilter allows https://poll-vote-app-frontend.onrender.com
    • Controller-level @CrossOrigin also whitelists http://localhost:4200 for dev
Frontend (Angular, standalone components)
  • Create polls with any number of options.
  • Vote on polls; counts are persisted and reflected in the UI.
  • CORS ready for separate FE/BE deployment
  • Environment-based API switching (local vs Render).
  • Simple, decoupled design API and UI deploy independently.

Key Features

  • Create, toggle, and delete tasks (JPA + Thymeleaf forms).
  • Server-side rendering with Bootstrap UI.
  • Dockerized app for easy run in any environment.
