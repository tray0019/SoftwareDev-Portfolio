Overview

A simple real-time chat application built with Spring Boot, STOMP over WebSocket, and Thymeleaf. Users can join the chat, send messages, and see updates instantly in the browser. Focused on clean configuration and publish/subscribe messaging.

Architecture

  • WebSocketConfig sets up STOMP over WebSocket with /chat endpoint and SockJS fallback
  • Controller maps messages from /app/sendMessage and broadcasts to topic/messages.
  • Model defines ChatMessage with fields: id, sender, content.
  • View is a Thymeleaf page (chat.html) using Bootstrap, SockJS, and STOMP.js

Key Features

  • Real-time chat with WebSocket + STOMP
  • SockJS fallback for wider browser support
  • Simple publish/subscribe model (/app, /topic).
  • Server-side rendered UI with Bootstrap
  • Dockerized app for easy deployment in any environment
