Overview

A simmple task manager built with Spring Boot, Thymleaf, and MySQL. Users can create tasks, toggle completion, and delte entries. Focused on clean MVC separation and basic CRUD with JPA.

Architecture

  • Controller handles HTTP requests and return Thymeleaf views.
  • Service encapsulates business logic.
  • Repository uses JPA for persistence
  • Entity models the Task with fields: id, text, completed.

Key Features

  • Create, toggle, and delete tasks (JPA + Thymeleaf forms).
  • Server-side rendering with Bootstrap UI.
  • Dockerized app for easy run in any environment.
